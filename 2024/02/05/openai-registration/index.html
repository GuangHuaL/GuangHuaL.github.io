OpenAI账号免费注册

本文最后更新于 2024-02-05T13:05:21+08:00

这篇文章快速教会你如何注册一个 OpenAI 账号

1、申请海外邮箱

国内的邮箱都不再允许注册，建议可以点击大纲跳转到海外邮箱 proton 注册教程，这个是完全免费的，申请起来非常方便，其他海外邮箱，如 Gmail 也可以。如果您有自己的海外邮箱，可以跳过这步，直接去看第 2 步。

outlook/hotmail/live 这类微软账户不行基本上升级就会被封，应该是微软给 openAI 提供了邮箱的注册国家信息，为了不必要的损失，请不要用这类的邮箱升级 Plus 或者绑定 API。

2、登录 ChatGPT 官网

可以直接点击后面这个连接，这个连接可以直接跳转到登录的界面：https://chat.openai.com/auth/login

1、点击【sign up】进入到注册界面，填写上注册好的邮箱。

 

2、点击继续，填写好密码后，点击继续，出现下图界面邮箱会受到一个链接。

 

3、点击【Verify email address】会直接跳转到 ChatGPT 继续填写姓名、生日，然后【Agree】

 

4、填写完姓名之后就注册好了，然后回到 ChatGPT 的登录界面

选择是 ChatGPT 还是 API 就好了

 

OpenAI 的用户体系分为两大类：ChatGPT 用户和 API 开发者用户，两个体系是相互独立的，用户名密码是一致的，但是访问入口和支付都是独立的：

ChatGPT 用户的登录入口：https://chat.openai.com

API 开发者用户的登录入口：https://platform.openai.com

ChatGPT 用户获取 GPT-4 的使用权限，是通过购买 GPT Plus 会员，会员的收费标准是 20 美元/每月。购买 GPT Plus 后，可以再 chat 界面选择和 GPT4 进行聊天。升级教程可以跳转到ChatGPT Plus 快速订阅购买教程

API 开发者用户获取 GPT-4 API 的使用权限，是通过升级付费帐户。然后需要预充值 5 美元以上，才能开通 GPT-4 的访问权限。
